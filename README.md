## About urban planning

<details><summary></summary>
<p>

作为一门学科是典型的社会科学，且其实践主导的学科特点对地理学和社会学等学科的发展有着重要的承上启下的作用2.城乡是规划研究的主要对象，从理论上看，规划需要有科学分析性的工作和各种理论支撑的参考。从实践上看，规划需要满足甲方的需求，具体表现为土地空间的合理分配.但是目前看来规划学科体系散乱，拿来主义的现象，没有逻辑自洽的核心理论支撑。规划本质是利益的分配，但却用科学性来遮掩这一事实，令人失望，虽然说科学并非以证伪为核心.但有些规划理论不存在证伪的可能性（伪科学），规划作为典型的社会科学，标榜着科学性，城市规划是土地财政的技术工具，

规划的脱节在于过时的知识体系未能和实际接轨，而象牙塔的问题则是过于理想化，需要考虑平衡的东西太少，而规划目前最要命的“算账”极少出现在教学中，哪怕是研究生，也极少涉及到诸如拆赔补偿等东西，学生们在繁杂的学业中学习了大量的在实践中无用的知识，而最重要的实践环节往往是大量缺失的，加上大学阶段经过比较理想主义的规划教育体系培养，学生们抱着为社会做贡献，帮助人民提高生活品质的美好想法进入设计院和规划局，结果发现自己只是个无情的画图机器，成为了未来“劳动密集性”的画图工，象牙塔和现实的巨大落差导致了很多人对规划行业失望. 城市规划原理的相当一大部分理论早已过时，却仍然在规划教育中以相当重要的角色存在着.

城市设计的核心价值在哪里?能匹配上规划界自称其有的价值吗?城乡规划的科学性体现在哪里?城市设计论述或者修改时，其支撑性的数据是什么?怎么改、为什么这么改、改了之后又能够带来什么好处?是我们追求的靠近帕累托最优吗，还是说只需要符合其“规范”?

对于一个学科而言，首先就是确定学科的研究主体边界，规划就是研究编规划的主体，编规划的方法，规划传导和执行的过程，和规划最终的影响，在城乡这块大的蛋糕中，进行利益的最合理的分配，对城市进行功能的分派和收益的再分配，保持感性思维去思考各方面诉求，有要保持理智谋求城市利益的最大化，这就是规划师的定位。但是现在的学科重心到编规划的方法，过分强调技术路线，而规划的编制过程本身又是政治化的，所以这种所谓的工程技术完全就是假工程，硬是要把政治的结果套在工程科学的外壳下，导致学科研究的畸形，因为规划的研究结果是政治，必须纠正学科发展中的技术主义倾向，回归规划作为公共政策类社会科学的本质，核心是空间实践，应分为城市研究，和规划研究，城市研究主要是规划编制中的战略制定，研究城市的社会环境情况及相关战略，如产业选择，人口预测等，规划研究主要服务于规划编制，传导和执行的过程和效果，帮助优化规划的各类机制并评估规划的影响，明确规划的结构，城市研究无疑具有明显的跨学科性，研究结果在表面上可以被轻易的放进ppt而作为吹水依据，其为目前学科研究重点，但是城市研究在编制的过程中本应该是学科的，综合社会学，地理学，经济学等学科的专业人员共同决策，但我国规划院没有这个机制，也不想花这个时间，而我国城乡规划的培养体系不注重研究能力的培养，很难做到其他领域学科的知识专业性，显得弱势和边缘，因此必须加强规划研究的建设，充分发挥与行业实践紧密联系的实践特征，解决规划政策落地的实际问题，从实践导向出发，坚持规划是实践的学科这个核心特点，好不好看传导，硬不硬看落地，城市经济学牛儿满天飞，但实际怎么协调各类用地，怎么回应社区需求，和政府的诉求都是规划的工作，非要去和人家专业的毕业生抢城市研究的饭碗，被人围追堵截怪谁呢，规划学科的人没有现代化的学科圈子，建立共识，现实共同的利益护城河，学科知识壁垒，地理学，此外一个学科专业要避免边缘化绝对不仅仅是依靠加强学科建设，的所谓专业性，更是要明确权力边界，规划是实践的学科，规划师作为实践者，规划图则的编制本身就是规划师的培养必修，想要抓住城市设计，不能完全给建筑学拿掉，城市设计需要联合培养，但规划师不仅仅是画图的技术人员，规划师应有明显的分工，需要有背后的政策人员，应该是训练有素的社会科学学者，研究和优化规划的机制框架，时刻在各层级的规划中起决策作用，社区规划怎么回应基层需求，区域规划如何实现有效的联动合作等，以及规划结果的评估，造成的社会经济影响，关心政治，学科建设完全服务于项目盈利，学科理论没有突破，政治地位，实际上城乡规划不是如何避免边缘化的问题，而是从未进入过核心地带，没有上层建筑和学科基础，只有画图狗支起来的蒙古包，成不了高楼大厦，继续走技术主义的劳动密集性产业，只有死路一条。   打不过?那就加入吗?

### SPSS
SPSS（Statistical Product and Service Solutions，即社会产品及服务解决方案），是全球公认的最优秀的统计分析软件包之一。由于 SPSS软件不用基于一些较为繁琐、枯燥的语句和命令，只要用户具备一般的计算机和统计原理知识，就能够使操作者得到所需的统计分析结果，特别是非统计学专业人员的青睐。作为数理统计的基本分析方法，SPSS以其在试验分析中的应用会大大降低试验中处理大批量数据分析工作的难度，提高试验分析、预测的准确性和可靠性的特点，在医药学、人文社科和化工方面的应用较为广泛.

### Space Syntax
空间句法是1970年代由英国UCL巴雷特建筑学院比尔·希列尔（Bill Hillier）教授首先提出的 过去、现在、未来，“空间” 一直是规划与建筑最基本的设计资源，而对空间的运用一直是设计师们想要也需要的一种专业技能. 空间句法为此提供了一种有效的空间量化理论与方法.
Sdna是空间句法的前沿模型，要是想学sdna首先需要掌握空间句法，掌握基础的空间句法概念。 sdna作为一个工具包，需要导入至arctoolbox需要配合gis操作使用
国内研究论文有杨滔从基于空间句法的角度研究多尺度城市空间网络形态，描述城市空间网络形态的概念型框架,从聚集、分散以及尺度三个机制维度解释城市空间网络形态的几何特征3
《空间是机器》 《空间的社会逻辑》 《空间句法简明教程》深圳大学 《空间句法在中国》段进，比尔希利尔 《空间句法与城市规划》（东南大学出版社出版）

### AHP

层次分析法，简称AHP，是指将与决策总是有关的元素分解成目标,准则,方案等层次,在此基础之上进行定性和定量分析的决策方法.该方法是美国运筹学家匹茨堡大学教授萨蒂于20世纪70年代初,在为美国国防部研究”根据各个工业部门对国家福利的贡献大小而进行电力分配”课题时,应用网络系统理论和多目标综合评价方法,提出的一种层级理论.
罗钏雯，基于三维GIS的生态城市智能节点应用研究4.

#### Blender
